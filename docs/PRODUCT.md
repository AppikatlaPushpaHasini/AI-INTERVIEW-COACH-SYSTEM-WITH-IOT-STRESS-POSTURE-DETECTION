# Product Notes

PrepGenie is positioned as a startup-style interview intelligence product.

## Core Value

- help candidates practice better
- help admins and mentors review realistic session data
- connect software analytics with optional sensor telemetry

## Product Qualities

- multi-user account flow
- admin access control
- session history persistence
- downloadable professional report
- support inbox and owner contact flow
- live IoT readiness for demos and project showcases

## Realistic Constraints

- interview analysis is heuristic unless connected to a more advanced AI scoring backend
- IoT values depend on correct ESP32 wiring and Wi-Fi reachability
- support email delivery is inbox-based unless SMTP/email API is added later
